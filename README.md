Setup Instructions

Frontend
1) Navigate to client directory:
      cd client
2Install dependencies:
      npm i
3)Start development server:
      npm run dev



Backend
1)Navigate to server directory:
      cd server
2)Install dependencies:
      npm i
3)Start server:
      nodemon server



Important Note
.env file is not ignored in this repository
